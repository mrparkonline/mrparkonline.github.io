Data Structures

The Data Structures course focuses on expanding our basic python skills and learning how to represent data better. These skills are crucial for all programmers as they make their leap from beginner to intermediate programmers.


1: Matrices & List Comprehension

We are learning how to empower our list skills in this lesson.

2: Map & Filter

We are learning some built-in functions that help can us power up our iterable data.

3: Tuples

We are learning a data structure that is immutable like a string but behaves like a list.

4: Sets

We are learning an implementation of a mathematical concept that describes a collection of distinct object.

5: Dictionary

We are learning our final data structure that provides custom addresses to each item in a database.
