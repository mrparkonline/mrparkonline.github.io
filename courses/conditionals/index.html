Conditionals

The Conditional Unit focuses on learning how to control the flow of our programs and to make decisions in our programs. We will be learning how to construct sound Boolean expressions to help us implement if statements to solve more complex problems.


1: Boolean Data Type

The Boolean data type allow us to implement logic in our code by simplifying statements to be either True or False.

2: Boolean Operators

In this lesson, we will be talking about built-in Python operators that will give us a Boolean result.

3: Conditional: If Statements

If is a built-in keyword in Python that allows us to write conditional statements.

4: Binary Conditions

By adding in a new keyword, else, we can have two possible flow to our program.

5: Multiple Conditions

By adding in a new keyword, elif, we can have more than two possible flow to our program.

6: Nested Conditional Statements

Nesting is the act of placing programming statements inside another. We can nest if statements to create complex decision trees in our program.
