    Python Basics

    Python Basics Unit is all about learning the basic skills of programming. We will be covering the use of data types, variables, inputs, outputs, and built-in functions. By having good fundamentals, we will be able to equip ourselves with a solid foundation to learn more complex ideas in programming.

    Grade 10 Grade 11



    Python Conditionals

    The Conditional Unit focuses on learning how to control the flow of our programs and to make decisions in our programs. We will be learning how to construct sound Boolean expressions to help us implement if statements to solve more complex problems.

    Grade 10 Grade 11



    While Loops

    While Loop is our very first iteration technique in Python Programming. It is a conditional based loop that help us repeat a block of code. By learning how to implement a while loop, we will be taking our first step in learning about Algorithms.

    Grade 10 Grade 11



    For Loops

    For Loop is a sequence-based iteration technique. A for loop allow us to traverse many iterable datatypes with ease compared to a while loop. This helps us build a foundation into manipulating and analyzing Strings and Lists.

    Grade 11



    Strings

    Strings are our first collection-like Python data type / object that we are learning. It represents all alphanumeric and special symbols held as a collection of item. In this unit we learn how to traverse strings, manipulate strings, and format strings to solve problems.

    Grade 11



    Lists

    Lists are our first mutable data type. Lists allow us to create basic database to hold multiple items with mixtures of data. We will be focusing on algorithm design that involve lists.

    Grade 11



    Functions

    In Python, we can create functions to create our own code unit to help us implement algorithms. This is a good unit to revisit all we have learned so far because we are applying all our skills into a function. With functions, we will now have callable standalone algorithms much like how we call upon built-in functions to help our code.
